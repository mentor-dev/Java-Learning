### **My solution: [Link](./src/Solution.java)**  

```java
import java.util.Scanner;

public class Solution {
    public static void main(String[] args) {
        Scanner console = new Scanner(System.in);
        if (console.hasNext()) {
            System.out.println(console.next());
        }
    }
}
```
